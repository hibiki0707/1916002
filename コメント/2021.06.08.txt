コマンドcdはWindows/Linuxどこに行っても使うので、覚えておく

ぜんざいのカレントディレクトリ（フォルダ）を変更するためのコマンド
cd　フォルダ名で、カレントディレクトリがそのフォルダの中になる。
cd ..で、上のフォルダに移動する。
cd d:\GitHub\Portal2dなど「フルパス」も指定できる

※注意、ボリューム（SSD/HDD）をまたぐ場合は、cdだけでは移動できない
c:や、d:などで、ボリューム切り替えをやってあげないと、移動できない。

☆cdで、例えば今見ているエクスプローラーのフォルダに移動したい場合に
エクスプローラーからコピペして移動してもいいけども、その場合はエクスプローラー
の上の入力欄に”cmd"って書いたら、そこをカレントディレクトリとしたコマンド
ラインが立ち上がる。


バッチファイルを作るときの注意点
実行する基本的に止まらないので、実行する前に「命令はこれであってるか？」
「対象ファイルはこれでOK？」

なので、実行系のバッチファイルを動かす前に、まずechoで、対象ファイルが
「正しい対象のみ」になっているのを確認する。

それでも怖い場合は、「バックアップ」とっておく。

copyファイル名　backup\ファイル名
とする。

コマンドラインで、演算しようとする際の注意

set value=2021
で、valueに2021が入った変数が定義される。
なので、
echo %value%
とやると2021が出力される。

これに加算したいといって
set value=value+2
とやると
value+2
が出力される。これはコマンドラインがデフォでは
「文字列演算」しかしない為です。これを数値として
演算したい場合は/aをsetの後に書く。

set valie=2021
set /a value=value+2
と書くとecho %value%で
2023が出力される。
